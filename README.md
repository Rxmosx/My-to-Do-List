# 🌌 My To-Do List — Dark Glassmorphism

Uma aplicação de gerenciamento de tarefas moderna e minimalista, desenvolvida com **React**. O projeto conta com uma interface futurista baseada na estética *Dark Glassmorphism* (efeitos de vidro translúcido, desfoques de fundo e neons sutis), oferecendo uma experiência de usuário fluida e reativa.

---

## 🛠️ Tecnologias Utilizadas

- **React** (Componentização, Hooks: `useState`, `useEffect`)
- **Vite** (Ambiente de desenvolvimento ultra veloz)
